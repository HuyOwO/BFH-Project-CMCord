# CMCord 

> Ứng dụng (MVP) nhắn tin và giao tiếp trực tuyến lấy cảm hứng từ Discord.

## Giới thiệu

CMCord là dự án môn học Công Nghệ Phần Mềm của nhóm sinh viên Đại học CMC.

Mục tiêu của dự án là xây dựng một hệ thống giao tiếp trực tuyến cho phép người dùng:

- Đăng ký và đăng nhập tài khoản
- Nhắn tin theo thời gian thực
- Tạo và tham gia server
- Tạo các kênh trò chuyện
- Quản lý hồ sơ cá nhân
- Kết nối và học tập cùng cộng đồng

## Mục tiêu MVP

Phiên bản MVP tập trung vào các chức năng cốt lõi:

- [x] Đăng ký/Đăng nhập tài khoản
- [x] Tạo Server
- [x] Tham gia Server
- [x] Tạo Channel
- [x] Tin nhắn&Thông báo thời gian thực
- [x] Chỉnh sửa hồ sơ cá nhân
- [x] Vai trò và quyền hạn người dùng
- [x] File sharing
